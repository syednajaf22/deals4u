Bulk Rewards Information
About Bulk Rewards

Use this feature to send the same reward to multiple users at once. Each user will receive an individual reward notification.

Benefits:
  • Save time by sending rewards in bulk
  • Ensure consistent reward amounts across users
  • Each user still receives a personalized notification
Make sure to review your selection before sending rewards. This action cannot be undone in bulk, though individual rewards can be deleted later.
